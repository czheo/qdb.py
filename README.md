# qdb.py
Python Implemented Quine DB

Inspired by https://github.com/gfredericks/quinedb

# Usage
### Print the source code
```
> python qdb.py
```

### Set key => value
```
> python qdb.py set quine db
```

### Get key
```
> python qdb.py get quine
db
```

### Flush DB
```
> python qdb.py flushdb
```
